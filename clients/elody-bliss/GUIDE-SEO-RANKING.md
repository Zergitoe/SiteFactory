# Guide SEO Ranking — Elody Bliss

> Genere le 2026-06-30. Suivre cette checklist dans l'ordre apporte 80% des resultats SEO d'un site local. Comptez 4-6 heures de travail au total, etalees sur 2-4 semaines. **Sans ces actions, meme un site parfait techniquement ne rank pas en local.**

## 1. Google Business Profile (PRIORITE #1 — fait en 30 minutes)

C'est le plus gros levier pour ranker en local sur "maquilleuse coiffeuse mariage Marseille". Une fiche bien optimisee fait apparaitre votre site dans le **pack Google Maps** (3 fiches au-dessus des resultats classiques).

**Creation** : https://business.google.com -> Gerer maintenant -> creer une fiche

**A renseigner integralement** :
- [ ] Nom exact : **Elody Bliss** (identique partout sur le web)
- [ ] Categorie principale : choisir la plus precise (ex: "Plombier" plutot que "Service de reparation")
- [ ] Adresse complete : Av. de Saint-Jérôme, 13013 Marseille, 13013 Marseille
- [ ] Telephone : 06 58 82 48 63
- [ ] Site web : https://https://elodybliss.fr/
- [ ] Horaires : Sur rendez-vous
- [ ] Zone de service : ajouter toutes les villes desservies (vous avez declare : Marseille, Aix-en-Provence, Bouches-du-Rhône)
- [ ] Description (750 caracteres) : integrer "maquilleuse coiffeuse mariage Marseille" naturellement
- [ ] Services : lister tous vos services avec descriptions (vous en avez 4 sur le site)
- [ ] Photos : minimum 10 photos (logo, devanture, equipe, realisations, interieur). Ajoutez-en 1 par semaine ensuite.
- [ ] Attributs : cocher tout ce qui s'applique (parking, accessibilite, paiement, etc.)
- [ ] Activer la messagerie + le bouton "Demander un devis"

**Verification** : Google envoie un code par courrier (5-14 jours). C'est obligatoire pour ranker.

## 2. Avis clients (objectif 20 avis dans les 3 mois)

Les avis sont le **2eme signal de ranking local** (apres GBP). Une fiche avec 20+ avis et une note >= 4.5 ranke largement au-dessus d'une fiche vide.

**Methode** :
- [ ] Apres chaque chantier/prestation, demander un avis par SMS avec le **lien direct** :
  https://search.google.com/local/writereview?placeid=VOTRE_PLACE_ID
  (recuperer le Place ID via : https://developers.google.com/maps/documentation/places/web-service/place-id)
- [ ] Texte SMS type : "Bonjour [Prenom], merci pour votre confiance ! Si tout s'est bien passe, un avis Google nous aiderait beaucoup : [LIEN]. Merci !"
- [ ] Repondre a TOUS les avis (positifs ET negatifs) sous 48h. Personnaliser la reponse.
- [ ] Ne JAMAIS acheter d'avis : Google les detecte et penalise (suspension de fiche).

**Cible realiste** : 1 avis sur 3 clients qui regoivent une demande. Pour 20 avis = 60 demandes.

## 3. Annuaires locaux et citations (1-2h)

Google utilise les **citations** (mentions de votre NAP : Nom-Adresse-Telephone) sur d'autres sites pour confirmer votre legitimite locale. Plus vous etes liste, plus vous gagnez en autorite.

**Inscriptions essentielles** (verifiez d'abord que vous n'y etes pas deja) :
- [ ] **Pages Jaunes** : pagesjaunes.fr/pro
- [ ] **Solocal** : solocal.com (gere les Pages Jaunes etendues)
- [ ] **118 712** : 118712.fr/professionnels
- [ ] **Yelp France** : biz.yelp.fr
- [ ] **Apple Plans** : mapsconnect.apple.com (souvent oublie, gros impact iPhone)
- [ ] **Bing Places** : bingplaces.com

**Annuaires sectoriels** (selon votre metier "Coiffure et maquillage de mariée") :
- [ ] Federation/syndicat de votre profession
- [ ] Chambre de Commerce ou Chambre des Metiers locale (souvent annuaire en ligne)

**REGLE D'OR** : votre **NAP** (Nom, Adresse, Telephone) doit etre **strictement identique** partout. Une seule incoherence (ex: "Tel : 06 12 34" vs "06.12.34") brouille Google.

## 4. Backlinks locaux (4-6h, etale sur 2 mois)

Les backlinks (liens entrants depuis d'autres sites) restent un facteur top 3 du ranking. **Qualite > quantite** : un lien depuis le site de la mairie vaut 100 liens d'annuaires generiques.

**Pistes faciles** (par ordre de difficulte) :
- [ ] **Site de votre mairie / communaute de communes** : sections "Commerces locaux" ou "Annuaire des entreprises". Demandez l'inscription par mail.
- [ ] **CCI / Chambre des Metiers** : fiche entreprise + lien si vous etes membre.
- [ ] **Federations professionnelles** : fiche annuaire des adherents.
- [ ] **Partenaires et fournisseurs** : echange de liens avec des metiers complementaires non concurrents (ex: plombier <-> electricien, coiffeur <-> esthetique).
- [ ] **Presse locale** : envoyer un communique de presse a [ville]-info, leprogres.fr, etc. lors d'un evenement (10 ans, nouveau service, prix recu).
- [ ] **Sponsoring local** : club de sport, ecole, association. Lien depuis leur site = excellent signal local.
- [ ] **Temoignages chez vos fournisseurs** : "Elody Bliss utilise nos produits" -> lien retour.

**A EVITER** :
- Achat de liens en masse (Fiverr, agences low-cost) : penalite Google quasi-garantie.
- Echanges de liens sur des sites hors-sujet.
- Annuaires generiques de mauvaise qualite (sites sans trafic).

## 5. Maintenance mensuelle (15 min/mois)

Le SEO local n'est pas un coup unique. Les fiches qui rankent sont celles qui **vivent**.

**Checklist mensuelle** :
- [ ] Ajouter 2-3 nouvelles photos sur GBP (chantiers, equipe, produits)
- [ ] Publier 1 "post" GBP (actu, promo, nouveau service) — visible 7 jours, signal de fraicheur
- [ ] Repondre a tous les avis recus
- [ ] Verifier que horaires/jours feries sont a jour
- [ ] Demander 5-10 nouveaux avis aux derniers clients

**Trimestrielle** :
- [ ] Verifier le ranking sur "maquilleuse coiffeuse mariage Marseille" en navigation privee
- [ ] Audit Search Console : pages sans clics, requetes a optimiser
- [ ] Mettre a jour les services / tarifs si evolution

## 6. Suivi des resultats

Outils gratuits a configurer immediatement (idealement deja fait par votre prestataire) :
- [ ] **Google Search Console** : search.google.com/search-console -> ajouter le domaine + soumettre le sitemap
- [ ] **Google Analytics 4** : analytics.google.com (deja integre si vous avez fourni l'ID GA4)
- [ ] **Google Business Profile Insights** : section "Statistiques" de votre fiche GBP

**Indicateurs cles a surveiller** :
- Nombre de vues fiche GBP (objectif : x2 en 3 mois)
- Nombre d'appels depuis GBP
- Position moyenne sur "maquilleuse coiffeuse mariage Marseille" (Search Console)
- Nombre d'avis Google (objectif : +5/mois)
